# Modular Inverse

$$3^{-1}\pmod{26}$$

| $$R_i$$ | $$D_i$$ |  | $$S_i$$ |
| :--- | :--- | :--- | :--- |
| 26 |  |  | 0 |
| 3 | 8 | $$3\times8+2=26$$ | 1 |
| 2 | 1 | $$2\times1+1=3$$ | $$0-1\times8=-8$$ |
| 1 |  |  | $$1-(-8)\times 1=9$$ |

* $$R_i$$is the reminder.
* $$S_i=S_{i-2}-S_{i-1}\times D_{i-1}$$
